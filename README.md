## Skills Built

### 🎬 tiktok-text-overlay
Add branded text overlays to TikTok videos — production-ready for TikTok Shop BOF campaigns.
- **PILL style** — Stacked deal announcement boxes (red + white)
- **HOOK style** — Large screen-filling hook text (@cakedfinds style)
- Pure Python + ffmpeg, no dependencies beyond standard tools
- Used in active TikTok Shop affiliate campaigns

### 📱 tiktok-draft
Post videos directly to TikTok Studio drafts via Playwright automation.
- Upload video + caption → appears in TikTok Studio ready to publish
- Supports @bestdealspaty197-style TikTok Shop accounts
- Bypasses unreliable Inbox API — uses session-authenticated browser automation

- Products: TIRTIR, Centella Serum, Medicube
- Fully automated from product JSON to ready-to-post draft
